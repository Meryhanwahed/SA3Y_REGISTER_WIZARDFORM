## Overview

This project is a multi-step registration form (wizard) developed for the Sa3y website. The form is fully responsive and built with a focus on user-friendly design and smooth functionality.


## Features

- **Technologies Used**:
  - **HTML5** for structure
  - **CSS3 & Bootstrap** for styling and layout
  - **JavaScript & jQuery** for interactive features and dynamic behavior
  - Fully responsive design to ensure compatibility across all devices

- **Key Components**:
  - Multi-step form with a clear progress indicator
  - User input validation and error handling
  - Smooth transitions between form steps for a better user experience
  - Mobile-first approach to ensure accessibility on smaller screens

## User Experience

The UI/UX design focuses on simplicity and clarity. The user-friendly layout, combined with responsive behavior, ensures that the form is easy to use whether on desktop, tablet, or mobile devices.
